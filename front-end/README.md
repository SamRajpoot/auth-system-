# React Frontend for Auth System

## Features
- Login (JWT)
- View activity logs (admin only)
- React Router for navigation
- Axios for API calls

## Setup
1. Run `npm install` in this folder.
2. Start the frontend: `npm start`
3. Backend should be running at `http://localhost:5000`

## Usage
- Go to `/login` to log in.
- Go to `/activity-logs` to view logs (admin only).

## Configuration
- API base URL is set to `http://localhost:5000/api` in `api.js`.
- Make sure CORS is enabled on the backend for `http://localhost:3000`.
